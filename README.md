# TandeHooks

Хуки для сайта работающие на технологии rabbitMQ

### Установка

```sh
$ npm install
```

Для установки среды можете воспользоваться nodeenv

```sh
$ pip install nodeenv
$ nodeenv env
$ source env/bin/activate
```

### Настройка

```s
DB_HOST=127.0.0.1
DB_PORT=3306
DB_USER=root
DB_PASS=
DB_NAME=
```

### Запуск

```sh
$ npm run start
```
